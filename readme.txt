Add a button and a shotcode to the Wordpress editor in order to easily embed a music score (sheet music or tabs) from jellynote.com

== Description ==

This plugin lets you add jellynote music score easily on your WordPress.

First, it adds a shortcode to the editor [jellynote url=""].

Second, a new button appears in the WordPress editor.

Third, you can configure the plugins in the options menu.

== Installation ==

1. Upload jellynote-viewer folder to the wordpress plugin folder (wp-content/plugins)
2. Activate the plugin through the 'Plugins' menu in WordPress

And that's all.

== Frequently Asked Questions ==

= What is Jellynote? =

Jellynote is a platform for sharing sheet music, tabs and covers. You can check it at jellynote.com
